# Open-Source Zone
GameMode From SAMP(San Andreas MultiPlayer) Open-Source Zone is a modified version of LinexZone's game mode. It's designed to be an optimized game mode, and I personally use it for testing purposes.  The game mode will be free to use, and I will be providing support and updates.

Development by Nextium.
